Building Software Intuitively

Elo is a systems compiled programming language designed to get out of your way and let you build reliable and robust software.

Getting started

The philosophy

Elo is designed so that developers focus more on what they are building rather than the language.

The language is called Elo because it means "chain link" in Portuguese which is closely related to this language's purpose as a tool. A strong and reliable piece of your project, but still intentional and intuitive.

We chose the slogan "Building software intuitively" to reflect the clear intent to be a robust but intuitive programming language.

Talk is cheap. Show me the code.

Development

Consider contributing

Elo is not a finished language. For now, Elo is nothing more than a prototype. But we are working hard to make it a real thing.

If you are interested in contributing to Elo's development, feel free to take a look at the official repository and contribute!
